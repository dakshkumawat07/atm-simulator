## ▶️ How to Use

### 1. Clone the repository

```bash
git clone https://github.com/dakshkumawat07/atm-simulator
```

### 2. Open the project folder

```bash
cd atm-simulator
```

### 3. Compile the program

Using GCC:

```bash
gcc main.c -o atm
```

### 4. Run the program

**Windows**

```bash
atm.exe
```

**Linux / macOS**

```bash
./atm
```

### 5. Use the ATM

Choose an option from the menu:

- **1** → Check current balance
- **2** → Deposit money
- **3** → Withdraw money
- **4** → Exit

Follow the on-screen instructions to complete your transaction.
