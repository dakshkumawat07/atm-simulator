# 🏧 ATM Simulator (C)

A ATM Simulator built in C as part of my C Programming learning journey. This project is developed step by step, with each version introducing new programming concepts while following professional GitHub practices.

---

## ✨ Features

- 💰 Check Account Balance
- ➕ Deposit Money
- ➖ Withdraw Money
- ✅ Deposit Validation
- ✅ Insufficient Balance Validation
- 🔐 PIN Authentication
- 🔒 Maximum 3 PIN Attempts
- 🚫 Card Blocking after 3 Incorrect Attempts
- 📋 Menu-Driven Interface
- 🎨 Improved Console UI
- ♻️ Reusable Functions
- ✅ Professional Transaction Messages

## ▶️ How to Use

1. Clone the repository

```bash
git clone https://github.com/dakshkumawat07/atm-simulator.git
```

2. Open the project folder

```bash
cd atm-simulator
```

3. Compile the program

```bash
gcc atm.c -o atm
```

4. Run the executable

### Windows

```bash
atm.exe
```

### Linux / Kali

```bash
./atm
```

5. Login using the default PIN

```
1234
```

6. Use the menu to:
- Check Balance
- Deposit Money
- Withdraw Money
- Exit
